Efficient face recognition using tensor subspace regression

1. Introduction

vector-based

Three representative appearance-based methods are Eigenface using principal component analysis (PCA), Fisherface using linear discriminant analysis (LDA), and Laplacian face using locality preserving project (LPP).

Both PCA and LDA are suitable for linearly embedded manifolds. However, it has been shown that face images may reside on a nonlinear submanifold. Laplacian face explicitly considers the manifold structure and generates nonlinear subspace embeddings which preserve the local structure of the face space. Laplacian face outperforms traditional PCA and LDA-based methods

tensor-based

Tensor Subspace Analysis (TSA) finds a projection from Rm⨂RnR^m \bigotimes R^nRm⨂Rn to Rm′⨂Rn′(m′<m,n′<n)R^{m'}\bigotimes R^{n'}\quad(m'Rm′⨂Rn′(m′<m,n′<n) which preserves the local structure of the face space (nonlinear submanifold). This structured dimensionality reduction has been shown to outperform traditional vector-based dimensionality reduction on face image data.

But the time complexity is high. The algorithm cannot be properly generalized to large scale training dataset.

2.1 Laplacian-based subspace learning

The Algorithm (Laplacian Eigenmap)

Given k points x1,...,xkx_1,...,x_kx1​,...,xk​ in RlR^lRl , we construct a weighted graph with k nodes, one for each point, and the set of edges connecting neighboring points to each other.

  1. Step 1 : (Constructing the Graph) We put an edge between nodes i and j if xix_ixi​ and xjx_jxj​ are “close” . There are two variations:

    1. ϵ\epsilonϵ - neighborhoods. (parameter ϵ∈R\epsilon\in Rϵ∈R) Nodes i and j are connected by an edge if ∣∣xi−xj∣∣2<ϵ||x_i-x_j||^2<\epsilon∣∣xi​−xj​∣∣2<ϵ .
      • Advantages: geometrically motivated, the relationship is naturally symmetric.
      • Disadvantages: often leads to graphs with several connected components, difficult to choose ϵ\epsilonϵ
    2. n nearest neighbors. (parameter n∈Nn \in Nn∈N) Nodes i and j are connected by an edge if i is among n nearest neighbors of j or j is among n nearest neighbors of i .
      • Advantages: simpler to choose, tends to lead to connected graphs
      • Disadvantages: less geometrically intuitive

  2. Step 2. (Choosing the weights) Here as well we have two variations for weighting the edges:

    1. Heat kernel. (parameter t∈Rt\in Rt∈R). If nodes i and j are connected. put

      Wij=e−∣∣xi−xj∣∣2tW_{ij}=e^{-\frac{||x_i-x_j||^2}{t}} Wij​=e−t∣∣xi​−xj​∣∣2​

    2. Simple-minded. (No parameters). Wij=1W_{ij}=1Wij​=1 if and only if vectors i and j are connected by an edge. A simplification which avoids the necessity of choosing t.

  3. Step 3. (Eigenmaps) Assume the graph G, constructed above, is connected, otherwise proceed with Step for each connected component. Compute eigenvalues and eigenvectors for the generalized eigenvector problem:

    Ly=λDyLy=\lambda Dy Ly=λDy

Laplacian is a symmetric, positive semidefinite matrix which can be though of as an operator on functions defined on vertices of G.

    Let y0,...,yk−1y_0,...,y_{k-1}y0​,...,yk−1​ be the solutions of equation (2), ordered according to their eigenvalues with y0y_0y0​ having the smallest eigenvalue (in face 0). 2.3 Tensor subspace analysis

TSA aims at finding a linear projective function which preserves the local geometrical structure of the data manifold from randomly sampled data points (face images) lying on this manifold. This leads to the following objective function:

min⁡U,V∑ij∣∣UTXiV−UTXjV∣∣F2\min_{U,V}\sum_{ij}||U^TX_iV-U^TX_jV||^2_F U,Vmin​ij∑​∣∣UTXi​V−UTXj​V∣∣F2​

where ∣∣⋅∣∣F||\cdot||_F∣∣⋅∣∣F​ is the Frobenius norm of matrix. This objective function means if two data points XiX_iXi​ and XjX_jXj​ are similar (WijW_{ij}Wij​ is large). we should keep YiY_iYi​ and YjY_jYj​ close in the learned subspace. Define a diagonal matrix D with Dii=∑jWijD_{ii} = \sum_jW_{ij}Dii​=∑j​Wij​ .

In addition to preserving the local geometrical structure. TSA also maximize the global variance for the data manifold in the new subspace in order to keep data points separable.

2.4 Regression for dimensionality reduction

Suppose we have m data points {xi}i=1m⊂Rn\{x_i\}^m_{i=1}\subset R^n{xi​}i=1m​⊂Rn and X=[x1,...,xm]X=[x_1,...,x_m]X=[x1​,...,xm​] . The projective function is denoted by a n×kn\times kn×k matrix AAA and the target point of xix_ixi​ is yi=ATxiy_i=A^Tx_iyi​=ATxi​ . Under the graph embedding framework, the objective function is

max⁡Atr(ATXWXTA)tr(ATXWXTA)\max_A\frac{tr(A^TXWX^TA)}{tr(A^TXWX^TA)} Amax​tr(ATXWXTA)tr(ATXWXTA)​

where WWW is an m×mm\times mm×m weighted adjacency matrix corresponding to a graph G which describes certain statistical or geometric properties of the dataset and D is a diagonal matrix with Dii=∑jWijD_{ii}=\sum_jW_{ij}Dii​=∑j​Wij​ .

Solving this optimization problem directly is time consuming. We first solve the follow optimization problem:

max⁡Ytr(YTWY)tr(YTDY)\max_Y\frac{tr(Y^TWY)}{tr(Y^TDY)} Ymax​tr(YTDY)tr(YTWY)​

where YYY is an m×nm\times nm×n matrix containing yyy's as row vectors (Y=XTAY=X^TAY=XTA) . Once yyy's are obtained, the optimal AAA can be computed by solving an efficient least square regression problem

min⁡A∑i=1m∣∣ATxi−yi∣∣2\min_A\sum_{i=1}^m||A^Tx_i-y_i||^2 Amin​i=1∑m​∣∣ATxi​−yi​∣∣2

We can apply the idea to second order tensor spaces. Extend SR to TSR from the following two aspects:

  1. The original SR learns coefficients for a linear model, while TSR learns coefficients for a multi-linear model
  2. SR’s regression problem has a closed form solution. However, we cannot derive closed form solutions directly for those two matrix (V and U). We develop an iterative optimization approach to solve the regression problem in TSR
