Spectral Regression: A Regression Framework for Efficient Regularized Subspace Learning

1. Introduction

Various manifold learning techniques, such as IOSMAP, Locally Linear Embedding(LLE) and Laplacian Eigenmap have been proposed which reduce the dimensionality of a fixed training set in a way that maximally preserve certain inter-point relationships.

  • LLE and Laplacian Eigenmap are local methods which attempt to preserve local geometry of the data
  • ISOMAP is a global method which attempts to preserve geometry at all scales.
  • One of the major limitations of these methods is that they do not generally provide a functional mapping between the high and low dimensional spaces that are valid both on and off the training data.

2. Graph Embedding View of Subspace learning

Let x1,...,xnx_1,...,x_nx1​,...,xn​ be the n data points sampled from an underlying submanifold MMM embedded in RmR^mRm , dimensionality reduction aims at finding {zi}i=1n⊂Rd,d<<m\{z_i\}_{i=1}^n\subset R^d,d<{zi​}i=1n​⊂Rd,d<<m , where ziz_izi​ can "represent " xix_ixi​ . Despite the different motivations of algorithms which have been proposed to solve this problem, they can be interpreted in a general graph embedding framework .

2.1.1 Locally Linear Embedding

The basis idea of LLE is that the data points might reside on a nonlinear submanifold, but it might be reasonable to assume that each local neighborhood is linear. Thus we can characterize the local geometry of these patches by linear coefficients that reconstruct each data point from its neighbors.

We first construct a k nearest neighbor graph G with weight matrix M. Reconstructing errors are measured by the cost function:

ϕ(M)=∑i=1n∣∣xi−∑j=1nMijxj∣∣2,s.t.∑j=1nMij=1\phi(M)=\sum_{i=1}^n||x_i-\sum_{j=1}^nM_{ij}x_j||^2,\quad s.t.\sum_{j=1}^nM_{ij}=1 ϕ(M)=i=1∑n​∣∣xi​−j=1∑n​Mij​xj​∣∣2,s.t.j=1∑n​Mij​=1

Consider the problem of mapping the original data points to a line so that each data point on the line can be represented as a linear combination of its neighbors with the coefficients MijM_{ij}Mij​ . Let y=(y1,y2,...,ym)Ty=(y_1,y_2,...,y_m)^Ty=(y1​,y2​,...,ym​)T be such a map. A reasonable criterion for choosing a “good” map is to minimize the following loss function:

Φ(y)=∑i=1n(yi−∑j=1nMijyj)2\Phi(y)=\sum_{i=1}^n(y_i-\sum_{j=1}^nM_{ij}y_j)^2 Φ(y)=i=1∑n​(yi​−j=1∑n​Mij​yj​)2

We fix the weights MijM_{ij}Mij​ here while optimizing the coordinates yiy_iyi​ .

Φ(y)=∑i=1n(yi−Miy)2=∑i=1n(yi2+(Miy)2−2yiMiy)=yTy+yTMTMy−2yTMy=yT(I−M)T(I−M)y\Phi(y) =\sum_{i=1}^n(y_i-M_iy)^2=\sum_{i=1}^n(y_i^2+(M_iy)^2-2y_iM_iy)\\ =y^Ty + y^TM^TMy-2y^TMy\\ =y^T(I-M)^T(I-M)y Φ(y)=i=1∑n​(yi​−Mi​y)2=i=1∑n​(yi2​+(Mi​y)2−2yi​Mi​y)=yTy+yTMTMy−2yTMy=yT(I−M)T(I−M)y

It can be shown that the optimal embedding yyy is given by the eigenvector corresponding to the minimum eigenvalue of the following eigen-problem:

(I−M)T(I−M)y=λy(I-M)^T(I-M)y=\lambda y (I−M)T(I−M)y=λy

where I is an m×mm\times mm×m identity matrix. Defind matrix WLLE=M+MT−MTMW_{LLE}=M+M^T-M^TMWLLE​=M+MT−MTM , We can rewrite the eigen-problem as:

(I−WLLE)y=λy  ⟹  WLLEy=(1−λ)y(I-W_{LLE})y=\lambda y\implies W_{LLE}y=(1-\lambda)y (I−WLLE​)y=λy⟹WLLE​y=(1−λ)y

Thus, the optimal embedding yyy is given by the eigenvector corresponding to the maximum eigenvalue of the eigen-problem:

WLLEy=λyW_{LLE}y=\lambda y WLLE​y=λy

2.1.2 ISOMAP

Let dMd_MdM​ be the geodesic distance measure on MMM and ddd the standard Euclidean distance measure in RmR^mRm . ISOMAP aims to find a Euclidean embedding such that Euclidean distances in RmR^mRm can provide a good approximation to the geodesic distances on MMM . That is

fopt=argmin⁡f∑i,j(dM(xi,xj)−d(f(xi),f(xj)))2f^{opt}=arg\min_f \sum_{i,j}(d_M(x_i,x_j)-d(f(x_i),f(x_j)))^2 fopt=argfmin​i,j∑​(dM​(xi​,xj​)−d(f(xi​),f(xj​)))2

In real life data set, the underlying manifold MMM is often unknown and hence the geodesic distance measure is also unknown.

  • We first construct a k nearest neighbor graph G over all data points to model the local geometry to discover the intrinsic geometrical structure of M .
  • The geodesic distances dM(i,j)d_M(i,j)dM​(i,j) between all pairs of points on the manifold M can be estimated by computing their shortest path distances dG(i,j)d_G(i,j)dG​(i,j) on the graph G.

The procedure is named Floyd-Warshall algorithm

  2. For each value of l=1,2,...,nl=1,2,...,nl=1,2,...,n in turn, replace all entries dG(xi,xy)d_G(x_i,x_y)dG​(xi​,xy​) by min⁡{dG(xi,xj),dG(xi,xl)+dG(xl,xj)}\min\{d_G(x_i,x_j),d_G(x_i,x_l)+d_G(x_l,x_j)\}min{dG​(xi​,xj​),dG​(xi​,xl​)+dG​(xl​,xj​)}.
  3. The matrix of final value DG={dG(xi,xj}D_G=\{d_G(x_i,x_j\}DG​={dG​(xi​,xj​} will contain the shortest path distances between all pairs of points in G.

2.1.4 Graph Embedding

Given a graph G with n vertices, each representing a data point, let W be a symmetric n×nn\times nn×n matrix with WijW_{ij}Wij​ having the weight of the edge joining vertices i and j. The G and W can be defined to characterize certain statistical or geometric properties of the data set.

  • The purpose of graph embedding is to represent each vertex of the graph as a low dimensional vector that preserves similarities between the vertex pairs, where similarity is measured by the edge weight.

Let y=[y1,y2,...,yn]Ty=[y_1,y_2,...,y_n]^Ty=[y1​,y2​,...,yn​]T be the map from the graph vertices to the real line. yyy 的最优解是最小化

∑i,j(yi−yj)2Wij\sum_{i,j}(y_i-y_j)^2W_{ij} i,j∑​(yi​−yj​)2Wij​

∑i,j(yi−yj)2Wij=2yTLy\sum_{i,j}(y_i-y_j)^2W_{ij}=2y^TLy i,j∑​(yi​−yj​)2Wij​=2yTLy
