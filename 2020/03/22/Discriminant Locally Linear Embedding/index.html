3. Gait Recognition

3.1 Baseline Algorithm

gallery : 算法或系统已知的人的集合，probes :是需要算法判定的签名。 算法分为四个部分

  1. Semiautomatically defines bounding boxes around the moving person in each frame of a sequence
  2. Extracts silhouettes from the bounding boxes
  3. Computes the gait period from the silhouettes
  4. Performs spatial-temporal correlation to copute the similarity between two gait sequences.

3.1 Silhouette Extraction

在 [3] 的数据集中，bounding box 以通过人工划分。

定义 silhouette 是一个人的像素区域，在提取轮廓前，需要先对场景进行背景建模。在每张图片的位置 (x,y)(x,y)(x,y) 计算背景的 RGB 值数据。

  • 使用在每一帧的人为划分的 bounding box 外的像素值计算：每个像素位置RGB值的平均值 μB(x,y)\mu_B(x,y)μB​(x,y) 和协方差 ΣB(x,y)\Sigma_B(x,y)ΣB​(x,y)。
  • 在 bounding box 内每个像素位置，计算 RBG-空间中每个像素对象与前面计算的 estimated 平均背景值的距离。基于 马氏距离(Mahalanobis distance) 将像素点分为前景和背景

Mahalanobis distance

马氏距离可以视为欧氏距离的修正，修正了各个维度的尺度不一致引起的问题。

单个数据点的马氏距离：

DM(x)=(x−μ)TΣ−1(x−μ)(11)D_M(x)=\sqrt{(x-\mu)^T\Sigma^{-1}(x-\mu)}\tag{11} DM​(x)=(x−μ)TΣ−1(x−μ)​(11)

数据点x, y 之间的马氏距离为：

DM(x,y)=(x−y)TΣ−1(x−y)(12)D_M(x,y)=\sqrt{(x-y)^T\Sigma^{-1}(x-y)}\tag{12} DM​(x,y)=(x−y)TΣ−1(x−y)​(12)

其中 Σ\SigmaΣ 为多维随机变量的协方差矩阵，μ\muμ 为样本均值，若协方差矩阵是识别矩阵，马氏距离就变为了欧氏距离。
