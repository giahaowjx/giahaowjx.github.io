2.1 Interpolation

Interpolation 基于在不同取样值之间函数的变化是平滑的的假设。但是随着取样点间假设的变化平滑度不同，interpolation 方法也会不同。

线性Interpolation 之外这里不细讲，但对于许多现实问题，导数的突变是不可接受的。

  • Tangent （切线）continuity (C1C^1C1 continuous, first derivative continuous, quadratic interpolation)，使用三个样本点。
  • second derivative continuous cubic curve (C2C^2C2 continuous)，使用四个样本点

4.1 Discrete Fourier Transform

Discrete Fourier Transform or DFT is a technique that takes as input a periodic signal of in nite length and decomposes it into a set of sine and cosine waves which when combined (via a process called inverse DFT) would provide the periodic signal itself.

Let the input to the DFT be a signal xxx with NNN samples denoted by x[0,1,⋯ ,N−1]x[0,1,\cdots,N-1]x[0,1,⋯,N−1]. 4.1.1 Why Sine and Cosig Waves

The sine and cosine waves of di erent frequencies are completely independent of each other. In other words, none of these waves can be given by a linear combination of the others.

4.5 Extension for 2D Interpretation

  1. The zero frequency (or the DC component) is at the bottom center of the image.
  2. The orientation of the cosine wave can be from 0 to 180
  3. As expected, high gray values are at lower radius from the center denoting concentration of the most the energy of the signal at lower frequencies.
  4. It is almost impossible to make any sense out of the phaseplot
  5. Feature detection is usually done in the spatial domain and the phase plot is typically not interpreted for spectral analysis

We have seen that the amplitude and phase plots repeat themselves to infi nity in positive and negative directions as an even and odd function respectively. The same will be true in 2D. But now since we are dealing with 2D functions, the plot will repeat itself in each of the four quadrant directions 象限.

Chapter 5 Feature Detection

5.1 Edge Detection

使用两步来进行特征检测：

  1. Find all the pixels that are part of an edge, called edgels.
  2. Use methods to aggregate this edgels into longer edges, sometimes using a compact parametric representation.

  • Gradient Based Detectors

we can quantify the direction and the strength of the edge respectively by

θ=tan⁡−1gygx∣∣∇f∣∣=gx2+gy2(5.2/5.3)\begin{aligned} \theta & = \tan^{-1}\frac{g_y}{g_x}\\ ||\nabla f|| & =\sqrt{g^2_x+g^2_y} \end{aligned}\tag{5.2/5.3} θ∣∣∇f∣∣​=tan−1gx​gy​​=gx2​+gy2​​​(5.2/5.3)

  • 对基于梯度的检测器
  1. Step or ramp edge can trigger response on multiple adjacent pixels based on how smooth the step or ramp is
  2. the response of the gradient based operators is different for different direction edges. As a result, the gradient based operators can miss edges based on the thresholding value since the value favors some directions over others.

  • Laplacian operator

为了寻找 zero crossing：

  1. 先将卷积后的图片通过阈值保留接近零的数。
  2. 然后标记像素的周围是否包含正值和负值来表示该标记像素为zero crossing

该方法卷积前同样需要先使用高斯滤波器减少噪声，高斯滤波器与Laplacian operator结合之后被称为 Laplacian of Gaussian 或 LoG。

该方法的不足：亮度变化非常快的时候，会导致一个edge被检查出两个 (spurious edges)。Threshold 也会导致某一些无法被检测到，称为 missed edges

  • Canny Edge Detector

An optimal edge detector should:

  1. Good detection : the filter should respond to edges only and not noise
  2. Good location : the detected edge is near the true edge
  3. minimal response : the edge’s exact location is marked with a single point response

Canny proposed a four step method:

  1. Suppress noise using low pass filtering
  2. Compute gradient magnitude and direction images
  • standard Sobel operator
  1. Apply non-maxima suppression to the gradient magnitude image
  • Canny 算子边缘检测原理及实现
  1. Use hysteresis and connectivity analysis to detect edges
  • 定义两个阈值 L 和 H , L<HLL<H
  • 若像素点的梯度大于 H 被认为是强边缘像素
  • 若像素点的梯度小于L 被认为是非边缘像素
  • 若像素点的梯度位于L，H之间，被认为是弱边缘像素
  • If a weak edge pixel is connected to at least one strong edge pixel, then it is considered to be continuation of a strong edge and hence marked.

每个方法都有参数需要调整，这些参数对结果的影响非常大，包括 滤波器的大小，阈值的大小等。

5.3 Other Non-Linear Filters

  • median filter

像素值替换为周围区域像素值的中值，用来去除离群点。由于设备限制等导致图片上的坏点（全黑或全白），称为 slat-pepper noise，高斯滤波器对这种噪声效果不好。

中值滤波器是 order statistics filters 非线性滤波器的一种，除此之外还有最大值滤波器，最小值滤波器。

最小值和最大值滤波器的效果是侵蚀和膨胀，侵蚀是将高值压缩，是图片更加暗淡，膨胀则是扩大高值区域，是图片更亮。

6.8 Degrees of Freedom

自由度为可以改变的参数个数，对二维数据来说，考虑平移和旋转，自由度为3，平移为两个，旋转的参数为角度视为一个，所以刚体变换的自由度为3。

考虑仿射变换，有剪切和放缩，每种变换有两个可变参数，似乎可变参数为 7 个（三个为刚体变换），但是最多只有六个参数，所以是六个自由度。
