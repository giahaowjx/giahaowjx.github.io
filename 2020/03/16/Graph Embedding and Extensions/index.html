1. Graph Embedding : A General Framework for Dimensionality Reduction

1.1 Graph Embedding

对一个通常的分类问题，模型训练的样本集表示为矩阵 X=[x1,x2,⋯ ,xN],xi∈RmX=[x_1,x_2,\cdots,x_N],x_i\in\Bbb{R}^mX=[x1​,x2​,⋯,xN​],xi​∈Rm ，其中 N 表示样本数，m 表示特征的维度。对监督学习问题，样本 xix_ixi​ 的类别标签为 ci∈{1,2,⋯ ,Nc}c_i\in\{1,2,\cdots,N_c\}ci​∈{1,2,⋯,Nc​}，其中 NcN_cNc​ 是类别数量。使用 πc\pi_cπc​ 和 ncn_cnc​ 分别表示属于类别 c 的下标集合及样本数。

在实际中，特征维度 m 通常非常大，所以需要将初始的高维空间映射到低维空间来减轻维度灾难。降维的任务即找到一个映射函数 F:x→y^F:x\rightarrow \hat{y}F:x→y^​ 将 x∈Rmx\in\Bbb{R}^mx∈Rm 映射到 y^∈Rm′\hat{y}\in\Bbb{R}^{m'}y^​∈Rm′ 其中 m>>m′m>>m'm>>m′

从图嵌入角度看，G={X,W}G=\{X,W\}G={X,W} 表示一个无向加权图，它的顶点集合为 XXX ，相似矩阵为 W∈RN×NW\in\Bbb{R}^{N\times N}W∈RN×N。实对称矩阵 W 的每一项表示一堆顶点的相似度，该值可能为负。 该矩阵可以使用不同的相似度计算的得到

  • 欧氏距离的高斯相似度：exp⁡(−∣∣xi−xj∣∣2/2σ)\exp(-||x_i-x_j||^2/2 \sigma)exp(−∣∣xi​−xj​∣∣2/2σ)

Kernelization

核函数可以直接用来拓展线性投影使之可以处理非线性问题。直观上看，核函数就是将数据从出事的 输入空间 映射到另一个 更高维的 希尔伯特空间 ϕ:x→F\phi:x\rightarrow \mathcal{F}ϕ:x→F 然后在映射后的空间使用线性算法。这个方法适用于那些仅需要计算 数据对内积 k(xi,kj)=ϕ(xi)⋅ϕ(xj)k(x_i,k_j)=\phi(x_i)\cdot\phi(x_j)k(xi​,kj​)=ϕ(xi​)⋅ϕ(xj​) 的算法。假设映射方向 w=∑iαiϕ(xi)w=\sum_i\alpha_i\phi(x_i)w=∑i​αi​ϕ(xi​) 且 K kernel Gram matrix Kij=ϕ(xi)⋅ϕ(xj)K_{ij}=\phi(x_i)\cdot\phi(x_j)Kij​=ϕ(xi​)⋅ϕ(xj​)，目标函数为

将训练集表示为 {Ai∈Rm1×m2×⋯×mn,i=1,2,⋯ ,N}\{\mathcal{A}_i\in\Bbb{R}^{m_1\times m_2\times\cdots\times m_n},i=1,2,\cdots,N\}{Ai​∈Rm1​×m2​×⋯×mn​,i=1,2,⋯,N} 投影后得张量维度更小，假设投影为标量

1.2 General Framework for Dimensionality Reduction

PCA

PCA 寻找使方差最大的投影方向，换句话说就是将去除方差最小的投影方向。

w∗=arg⁡min⁡wTw=1wTCwC=1N∑i=1N(xi−xˉ)(xi−xˉ)T=1NX(I−1NeeT)2XT=1N∑i=1N(xi−xˉ)(xi−xˉ)T=1NX(I−1NeeT)XT\qquad \qquad \qquad w^*=\arg\min_{w^Tw=1}w^TCw\\ C=\frac{1}{N}\sum_{i=1}^N(x_i-\bar{x})(x_i-\bar{x})^T=\frac{1}{N}X(I-\frac{1}{N}ee^T)^2X^T\\ =\frac{1}{N}\sum_{i=1}^N(x_i-\bar{x})(x_i-\bar{x})^T=\frac{1}{N}X(I-\frac{1}{N}ee^T)X^T w∗=argwTw=1min​wTCwC=N1​i=1∑N​(xi​−xˉ)(xi​−xˉ)T=N1​X(I−N1​eeT)2XT=N1​i=1∑N​(xi​−xˉ)(xi​−xˉ)T=N1​X(I−N1​eeT)XT

  • 其中的平方项展开后与不加平方相同！！

其中 eee 使一个 N-维向量，I 是一个识别矩阵，C 是协方差矩阵，xˉ\bar{x}xˉ 是所有样本的均值。

  • PCA 遵循图嵌入的线性化，给予所有数据对相同的权重，限制投影向量的长度为 1

KPCA 是在 PCA 上使用了 核函数 策略，2DPCA 是 PCA 的一个简化二位张量化（只对一个模态进行投影）。PCA 可以与 LPP 相关，但是不能完全将PCA说成LPP框架的特例，因为PCA使用了一个最大化指标，而LPP是基于最小化。
